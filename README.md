Installation
Update your userscript manager with the new version. No manual configuration required - all improvements are automatic.

Compatibility
Browsers: All modern browsers with userscript support
MangaDex: Compatible with current MangaDex.org structure
Script Managers: Tampermonkey, Violentmonkey, Greasemonkey
Dependencies: None (pure JavaScript)
